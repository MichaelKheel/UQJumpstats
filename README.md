# Unique Jumpstats plugin

Jumping statistics plugin for Goldsrc Engine (Counter-Strike 1.6) originally developed by Borjomi, but heavily modified by me.

### Supported Build
* ReHLDS 3.4.0.X
* Linux 5787 and higher
* Windows 5758 and higher

### AMXMODX Version
* 1.8.0+

### Required modules
* amxmodx
* amxmisc
* cstrike
* engine
* fakemeta
* hamsandwich
* mysql
